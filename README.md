# chrome-text-input-hold
ブラウザ上でテキストまたはテキストエリアに入力した文章の履歴を保持し、履歴からコピーを可能とする
※履歴は最大100件保持

# How to use
1. ブラウザ上でテキストまたはテキストエリアに入力
2. 拡張機能のアイコンクリック
3. 履歴から利用したいテキストをクリックでクリップボードにコピーされる
